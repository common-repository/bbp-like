== Description ==
Like for bbPress plugin will add a like button to bbpress topics and replies, bbpress forum members can like or vote topics and replies

When logged in users View bbpress forum topic, he can like / vote the bbpress topics or replies, If user clicked "like it now" button, the text of the button will be changed to "you liked it", and the score will +1.

== Installation ==
Like for bbPress is a simple & quick & light & powerful bbp solution, it is very easy to use and install, what you need is just 

1:Upload the Like for bbPress Plugin to your wordpress site
2:Activate it 
3:Please check topics in your bbPress forum, logged in user can like / vote the bbPress topics and replies
It will works well, that's all :)
